### Claim-types
- Framing existing views and how they operate with one another is an important part for intellectual inquiry
- Enthymeme
	- incomplete arguments
	- Claims with only partial reasoning
- Categorical
	- X is (or is not) Y
	- Example: America is a Christian country
- Definition
	- X is (or is not) Y, where Y is disputable
		- Example: WashU is an elite institution
- Causal
	- X is (or is not) the consequence of Y
- Resemblance
	- X is (or is not) like Y
- Evaluative
	- X is (or not) a good Y.
- Proposal
	- X should (or should not) do Y
